# Getting Started  
## Prerequisites
The only dependency for BLISTR is zlib, which is used for handling gz-compressed files.  
## Installation  
```
git clone https://github.com/Wenfei-Xian/BLISTR.git  
cd BLISTR/src  
make
```
## Installation with Bioconda  
`conda install BLISTR`
